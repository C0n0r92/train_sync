## Week 2 Bugs Found & Fixed (Backend API)

| Bug | Root Cause | Fix | Impact |
|-----|-----------|-----|--------|
| JWT Secret API Error | Rails 5 vs 6+ incompatibility | `secrets.secret_key_base` → `config.secret_key_base` | Would block all JWT generation |
| Empty Blocks Parsing | Rails form parameter handling | Added normalization: `[] if blank?` | Created invalid JSONB data |
| Double Render in Auth | Missing return statement | Added `return` after render in auth methods | Returned 500 instead of 403 |
| Generic Error Messages | No field-level validation | Parse validation errors from model | Poor API consumer experience |
